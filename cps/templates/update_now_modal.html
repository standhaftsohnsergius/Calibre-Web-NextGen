{{ _('Update Calibre-Web NextGen') }}

{{ _('Updating pulls the new image and recreates the container. It is quick, and your library, settings and reading progress are safe — they live in your mounted volumes, not in the container.') }}

{{ _('How do you run Calibre-Web NextGen?') }}

{{ _('Run this from the folder that holds your compose file:') }}

docker compose pull calibre-web-nextgen && docker compose up -d calibre-web-nextgen

{{ _('Replace calibre-web-nextgen with your service name if it differs.') }}

{{ _('Pull the new image, then recreate your container with the same options you started it with:') }}

docker pull ghcr.io/new-usemame/calibre-web-nextgen:latest

{{ _('Then stop and remove the old container and start a new one with your usual docker run command. Your mounted volumes keep all data.') }}

  1. {{ _('Open the Docker tab in the Unraid web UI.') }}
  2. {{ _('Click the Calibre-Web NextGen container icon and choose Force update (check for updates first if you want to see what changed).') }}

{{ _('For hands-off updates, install the "CA Auto Update Applications" plugin from Community Applications.') }}

{{ _('Portainer:') }} {{ _('Images → pull the image below, then Containers → select Calibre-Web NextGen → Recreate with "Re-pull image" enabled.') }}

{{ _('Synology Container Manager:') }} {{ _('Registry → download the image below (tag: latest), then stop the container → Action → Reset to recreate it.') }}

ghcr.io/new-usemame/calibre-web-nextgen:latest
