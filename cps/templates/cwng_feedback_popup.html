{# ──────────────────────────────────────────────────────────────────────────── CWNG #60 — "Back to the classic view" feedback popup. Shown on the CLASSIC (legacy) page when a user has just switched back from the new UI. The new-UI TopBar "Back to the classic view" item navigates here with ?cwng_feedback=newui; this partial's script detects that marker, shows a short, two-step, fully anonymous feedback prompt, then strips the marker from the URL. Anonymity ("unmarked mail"): only { type, reasons, comment } is POSTed to the feedback endpoint, which stores nothing identifying — no account, name, IP, version or device. The connection IP terminates at Cloudflare's edge and is used only as a short-lived one-way hash for spam rate-limiting, never stored. See notes/FEEDBACK-SERVER-DESIGN.md. caliBlur-safe: every class is cwng-fb-* and styles are scoped here — caliBlur.css hijacks Bootstrap .alert/.modal into fixed toasts, so we use none of those. ──────────────────────────────────────────────────────────────────────────── #}
